# VideoRetalking — 22 проблемных видео

## Описание проблемы

При генерации 1000 видео датасета VideoRetalking_FFpp_vox2, 22 видео не удалось сгенерировать из-за ошибок распознавания лиц. VideoRetalking требует, чтобы лицо было обнаружено на **каждом кадре** входного видео (`face_detect()` в `utils/inference_utils.py`). Если хотя бы на одном кадре детектор возвращает `None`, весь видеоролик падает с ошибкой.

## Ошибки

- `UnboundLocalError: local variable 'mask_sharp' referenced before assignment` — face parsing не может вычислить маску, т.к. лицо не найдено
- `UnboundLocalError: local variable 'lm' referenced before assignment` — landmark extraction не получил landmarks, т.к. face detection вернул None

## План исправления

1. Для каждого из 22 видео — просканировать все кадры face-детектором
2. Найти самый длинный непрерывный отрезок с обнаруженным лицом (минимум 5 секунд)
3. Обрезать видео (ffmpeg) до этого отрезка
4. Перезапустить VideoRetalking на обрезанных версиях
