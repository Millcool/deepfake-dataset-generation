Регистрация внешнего детектора

Подключите свой детектор дипфейков к нашей системе оценки

API-контракт

Ваш детектор должен принимать видеофайл и возвращать оценку подлинности.

Запрос (что мы отправляем)

POST <your_api_url> Content-Type: multipart/form-data file: <video.mp4> (binary)

Ответ (что должен вернуть детектор)

{ "score": 0.85, // float [0,1], REQUIRED "label": "fake", // string, optional "confidence": 0.85, // float [0,1], optional "metadata": {} // object, optional }

Поле score — единственное обязательное. Значение 0 = real, 1 = fake. Этого достаточно для AUC, ROC, confusion matrix, score distributions, EER.
